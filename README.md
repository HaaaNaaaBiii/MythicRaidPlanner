## 介面總覽

程式啟動後的介面分為四個區域：

```
┌─────────────────────────────────────────────────────────┐
│  Row 1: 我的團隊 WCL  [Report ID] [取得首領列表] [載入陣容]      │
│  Row 2: 參考 WCL       [Report ID] [取得首領列表] [導入承傷數據]  │
│  Row 3: [儲存方案] [載入方案] [匯出MRT Note]        版本資訊     │
├──────────────┬──────────────────────────────────────────┤
│  團隊陣容     │  承傷圖表 (可拖動調整大小)                       │
│  (Roster)     │  ─ X軸: MM:SS 時間格式                          │
│               │  ─ 十字準線跟隨滑鼠                              │
│               │  ─ Boss技能標籤 (橘色)                           │
│               │  ─ 治療技能標籤 (職業顏色)                       │
│               ├──────────────────────────────────────────┤
│  治療大招分配  │  首領技能時間軸                                  │
│  (CD分配表)   │  ─ 點擊可在圖表上標記                            │
│               │                              [清除標記]          │
└──────────────┴──────────────────────────────────────────┘
```

所有區塊之間的分隔線皆可 **拖動調整大小**。

## 基本操作流程

### 第一步：載入資料
1. 在 **我的團隊 WCL** 輸入你的 Report ID → 點擊 **「取得首領列表」**
2. 從下拉選單選擇要規劃的 Boss → 點擊 **「載入陣容」**
3. 左側會顯示團隊陣容，左下會自動填入所有治療/減傷技能

### 第二步：載入參考資料（可選）
1. 在 **參考 WCL** 輸入別人的 Report ID → 點擊 **「取得首領列表」**
2. 選擇相同 Boss → 點擊 **「導入承傷數據」**
3. 右側圖表會更新為參考團隊的承傷曲線

### 第三步：安排減傷
- **方法 A — 圖表點擊分配（推薦）**：
  1. 在承傷圖表上 **左鍵點擊** 你想安排減傷的時間點
  2. 彈出選單列出所有當前可用的技能（冷卻中的會自動隱藏）
  3. 點擊技能名稱即可分配
- **方法 B — 手動輸入**：
  1. 直接在左下角 CD 分配表的「分配時間」欄位輸入時間
  2. 格式：`MM:SS`（例如 `01:30`）
  3. 同一技能多次使用以逗號分隔：`01:30, 04:00`

### 第四步：匯出
- 點擊 **「匯出 MRT Note」** → 自動複製到剪貼簿
- 到遊戲中貼到 MRT Note 即可

## 功能詳細說明

### 載入 WCL 資料

| 按鈕 | 說明 |
|------|------|
| 取得首領列表 | 從 WCL 讀取所有 Boss 戰鬥（自動過濾小怪 Trash） |
| 載入陣容 | 讀取團隊成員、治療職業，自動填入左側面板 |
| 導入承傷數據 | 讀取參考 WCL 的承傷圖表與 Boss 技能時間軸 |

- 首領列表會顯示難度（普通/英雄/傳奇）與戰鬥結果（擊殺/滅團）
- Trash Fights 會自動被過濾掉

### 承傷圖表互動

#### 十字準線
滑鼠移動時，圖表上會顯示金色十字準線，即時顯示：
- 當前時間（MM:SS 格式）
- 當前傷害數值

#### Boss 技能標籤（橘色）
圖表上會自動顯示造成團隊傷害的 Boss 技能，包含：
- 分攤、團傷、團傷DOT、範圍傷害、滅團、治療吸收 等
- 純機制類（躲避、打斷、坦克、小怪等）不會顯示在圖表上
- 標籤 Y 軸位置會對齊該時間點的實際傷害值

#### 治療技能標籤（職業顏色）
分配減傷後，圖表上會顯示：
- **水平色條**：表示技能持續時間（起點到結束）
- **文字標籤**：顯示「玩家名 技能名」
- 顏色對應職業（如牧師白色、聖騎粉色、薩滿藍色）
- 多個技能重疊時會自動錯開

#### 左鍵點擊分配
在圖表上左鍵點擊會彈出可用技能選單：
- 只顯示當前時間點可用的技能
- 冷卻中的技能會自動隱藏
- 點擊即分配，時間會附加到左下角表格

### 首領技能時間軸

- 連續施放的相同技能會自動合併為時間範圍（`01:30 - 01:45`）
- 技能名稱後方會標註機制類型（如 `[分攤]`、`[團傷]`、`[躲避]`）
- 時間軸為**唯讀**，不可編輯

#### 點擊標記（Toggle）
- **點擊一次**：在圖表上畫出青色虛線標記 + 技能名稱
- **再點擊一次**：移除標記
- 時間範圍技能會畫出起點（粗虛線）和終點（細點線）

#### 清除標記
- 點擊「清除標記」按鈕可一次清除所有標記
- 若有已分配的減傷時間，會彈出確認對話框
- 確認後會同時清除所有分配時間

### 治療大招分配

程式會自動為團隊中的治療和減傷職業生成技能列表：

| 職業 | 技能 | CD | 持續 |
|------|------|-----|------|
| 牧師 | 神聖讚美詩 | 180秒 | 8秒 |
| 牧師 | 群體驅散 | 120秒 | 1秒 |
| 聖騎士 | 光環精通 | 180秒 | 8秒 |
| 薩滿 | 靈魂鏈接圖騰 | 180秒 | 6秒 |
| 薩滿 | 治療之潮圖騰 | 180秒 | 10秒 |
| 德魯伊 | 寧靜 | 180秒 | 6秒 |
| 德魯伊 | 奔竄咆哮 | 120秒 | 8秒 |
| 武僧 | 還魂術 | 180秒 | 6秒 |
| 喚魔師 | 倒轉 | 240秒 | 4秒 |
| 死亡騎士 | 反魔法護罩 | 240秒 | 6秒 |
| 戰士 | 命令之吼 | 180秒 | 10秒 |

- 同一技能可分配多個時間（逗號分隔）
- 系統會自動計算冷卻，避免重複分配

### 個減

- 左下角表格最後一列為 **「個減」**（個人減傷）
- 不綁定任何玩家，時間格式同其他技能
- 匯出 MRT Note 時只輸出 TTS 提醒，不輸出 spell ID
- 用途：安排特定機制需要全團開個人減傷的時間點

### 儲存與載入方案

#### 儲存方案（💾）
將當前所有資料存為 `.json` 檔案，包括：
- 我的團隊 / 參考團隊的 Report ID 與 Boss 選擇
- 團隊陣容
- 所有治療大招分配時間
- Boss 技能時間軸
- 圖表數據

#### 載入方案（📂）
從 `.json` 檔案還原所有狀態，包括圖表和分配。

### 匯出 / 匯入 MRT Note

#### 匯出格式
```
{time:01:30} 玩家A 神聖讚美詩 {spell:64843}
{time:02:30} 玩家B 靈魂鏈接圖騰 {spell:98008}
{time:03:00} 玩家C 反魔法護罩 {spell:51052}
{time:04:00}  個減
```

- 格式：`{time:MM:SS} 玩家名 技能名 {spell:ID}`
- 個減無 spell ID，玩家欄為空
- 所有條目按時間排序
- 點擊「匯出 MRT Note」後自動複製到剪貼簿，可直接貼入遊戲 MRT 插件

#### 匯入
- 點擊「匯入 MRT Note」後貼上 MRT Note 文字
- 格式：`{time:MM:SS} 玩家名 技能名 {spell:ID}`
- 系統會自動比對玩家名和 spell ID，將時間填入對應的分配欄位
- 匯入後會顯示成功比對的數量

## 常見問題

**Q: 為什麼圖表上只顯示部分 Boss 技能標籤？**  
A: 只顯示會造成團隊傷害的技能（分攤、團傷等），純機制類（躲避、打斷、坦克等）不顯示，以保持圖表清晰。

**Q: 同一個技能可以用兩次嗎？**  
A: 可以。在分配時間欄用逗號分隔填入多個時間（如 `01:30, 04:00`），系統會自動檢查冷卻是否允許。

**Q: 清除標記會清掉已分配的時間嗎？**  
A: 會。如果有已分配的時間，點擊「清除標記」時會跳出確認對話框。
